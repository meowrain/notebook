# CSS-相对单位长度与绝对单位长度

## 相对单位长度

> 相对单位长度，比如 `em` 和 `rem`，它们的实际值会依赖其他长度的值而决定。 比如 `em` 的大小基于元素字体的字体大小。 如果使用它来设置 `font-size` 值，它的值会跟随父元素的 `font-size` 值来改变。
